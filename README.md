# NecoClicker

Лёгкий автокликер для Windows с глобальными хоткеями, макросами,
пошаговой записью точек и мини-режимом для игр.

![icon](build/appicon.png)

## Возможности

### Кликеры

- **Простой кликер** — клик по фиксированным координатам или по текущему
  курсору с заданным интервалом. Поддерживается ЛКМ / ПКМ / СКМ /
  Mouse4 / Mouse5. Несколько профилей со своими хоткеями, активный
  отмечается ⭐.
- **По точкам** — пошаговый рекордер: наведи курсор и нажми клавишу
  (по умолчанию F10). Записывается до 10 точек. У каждого шага можно
  выбрать кнопку. Запускается циклом с настраиваемой задержкой.
- **Макросы** — цепочки действий (клик / задержка / перемещение / drag)
  с произвольным порядком, повторами и хоткеем на каждую цепочку.
- **Кликер с таймером** — автостоп по времени, числу кликов или обоим.
- **Хаотичный кликер** — `±N мс` к интервалу на каждый клик
  (анти-детект "ровного ритма" в играх).

### Точность кликов

- Прямой `user32!SendInput` без CGO-обёрток.
- Поддержка sub-миллисекундных интервалов и режима 0 мс (tight loop).
- Глобальное случайное смещение клика **±N px** (Settings → Поведение).
- Per-Monitor V2 DPI awareness — координаты не плывут на HiDPI.

### Хоткеи

- Глобальные через WH_KEYBOARD_LL + WH_MOUSE_LL хуки.
- Поддерживаются все обычные клавиши, F1–F24 и Mouse4 / Mouse5.
- **Hotkey recorder** — кнопка «Записать» вместо текстового ввода.
- Отдельный хоткей **паузы / возобновления** (по умолч. F8) — замораживает
  кликер на месте, не останавливая.

### Интерфейс

- 6 тем: light, dark, enemy-dark, purple-neon, green-neon, vampire.
  При первом запуске берётся системная.
- **Click-ping overlay** — прозрачные вспышки прямо на экране в точке
  клика. Цвет привязан к теме. Тогглится в Settings.
- **Мини-режим** — при включении «поверх окон» окно сжимается до
  узкой полоски с кнопками старт/стоп/пауза и именем активного
  профиля. Возврат к полному виду в один клик.
- **Tray** — крестик сворачивает в трей. ПКМ по иконке: показать
  окно, запустить активный профиль, поверх окон, выйти.
- **Тест** — большая зона замера с пульсами по каждому клику +
  CPS-метр (текущий / пик / всего) + bar-chart истории.

### Конфигурация

- Всё хранится в `%APPDATA%\NecoClicker\config.json`.
- Импорт / экспорт `.necoclicker.json` через Settings.

## Сборка

```bat
:: Тулчейн (winget)
winget install -e --id GoLang.Go
winget install -e --id OpenJS.NodeJS.LTS
winget install -e --id BrechtSanders.WinLibs.POSIX.UCRT
winget install -e --id UPX.UPX
winget install -e --id NSIS.NSIS

go install github.com/wailsapp/wails/v2/cmd/wails@latest
npm install -g pnpm

:: Сборка
build.bat
```

Результат — `build/bin/NecoClicker.exe` (~4.5 MB после UPX) и NSIS
установщик. Оба artefact'а standalone, кроме WebView2 runtime
(на Windows 10 1803+ и Windows 11 уже предустановлен).

## Хоткеи (формат)

`Ctrl`, `Alt`, `Shift`, `Win` + одна из:

- `A`–`Z`, `0`–`9`
- `F1`–`F24`
- `Space`, `Enter`, `Tab`, `Esc`, `Insert`, `Delete`,
  `Home`, `End`, `PgUp`, `PgDn`, `Up`, `Down`, `Left`, `Right`
- `Mouse4`, `Mouse5`

Примеры: `F6`, `Ctrl+Shift+F1`, `Alt+Q`, `Ctrl+Mouse4`.

## Известные ограничения

- Защищённые системой комбинации (`Win+L`, `Ctrl+Alt+Del`) не
  перехватываются — это by design Windows.
- Чтобы кликать в окно админ-приложения, NecoClicker нужно тоже
  запустить от админа (UAC isolation).
